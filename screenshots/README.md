# Screenshots

This folder contains visual documentation of the data cleaning, transformation, analysis, and dashboard development process for the Sales Data Cleaning and Analysis Project.

## Files

### 1. messy_data_sheet.png
Shows the original dataset before cleaning. The data contains quality issues such as:
- Missing values
- Inconsistent formatting
- Duplicate records
- Invalid entries
- Data entry errors

This serves as the starting point of the project.

---

### 2. power_query.png
Displays the Power Query workflow used to clean and transform the dataset.

Key transformations performed include:
- Standardizing data formats
- Handling missing values
- Correcting inconsistencies
- Removing duplicate records
- Preparing the dataset for analysis

This demonstrates the automation and repeatability of the cleaning process.

---

### 3. cleaned_data_sheet.png
Shows the final cleaned dataset after all transformations were applied.

Improvements include:
- Consistent formatting
- Validated categorical data
- Reduced errors
- Analysis-ready structure
- Improved data quality

This dataset serves as the foundation for reporting and dashboard creation.

---

### 4. analysis.png
Contains pivot tables and analytical summaries generated from the cleaned dataset.

The analysis was used to:
- Measure sales performance
- Compare regional performance
- Evaluate product contributions
- Assess order status distribution
- Generate business insights

---

### 5. dashboard.png
Displays the final interactive Excel dashboard created from the cleaned dataset.

Dashboard features include:
- Key Performance Indicators (KPIs)
- Product performance analysis
- Regional sales breakdown
- Order status analysis
- Interactive filtering through slicers

The dashboard provides a consolidated view of business performance and supports data-driven decision-making.

---

## Purpose

These screenshots document the complete workflow from raw data to actionable business insights, demonstrating practical skills in:

- Data Cleaning
- Power Query
- Data Analysis
- Dashboard Development
